Crystals Collector!

Instructions

  1. You will be given a random number at the start of the game.
  2. Using the $(this) keyword specifies that we should be extracting the crystal value of the clicked crystal.
  3. You win the game by matching your total score to random number.
  4. You lose the game if your total score goes above the random number.
  5. The value of each crystal is hidden from you until you click on it.
  6. Each time when the game starts, the game will change the values of each crystal.
